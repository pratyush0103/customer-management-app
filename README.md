# Demo Project

This repository contains a full-stack application with a Spring Boot backend and a Next.js(React) frontend.

## Project Structure

- **`demo-backend/`**: A Spring Boot application (Kotlin) serving the REST API.
- **`demo-frontend/`**: A Next.js application (React/TypeScript) for the user interface.

## Getting Started

### Backend
Navigate to the `demo-backend` directory and run:

```bash
cd demo-backend
./gradlew bootRun
```

The server will start on `http://localhost:8080`.

### Frontend
Navigate to the `demo-frontend` directory and run:

```bash
cd demo-frontend
npm install
npm run dev
```

The frontend will start on `http://localhost:3000`.
